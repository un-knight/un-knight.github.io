第五章：神经网络（Neural Networks）阶段二——学习训练

发表于 2017-01-16   |   分类于 Machine Learning   |     |   阅读次数

说明：这是我在学习吴恩达老师在 coursera 中 《Machine Learning》课程的过程中所做的笔记与总结，因为能力有限所以笔记中难免有错，欢迎交流指正。

损失函数

与逻辑回归相类似，神经网络的损失函数定义为：

$$h_{\Theta}(x) \in \mathbb{R}^{K}\ \ \ \ (h_{\Theta}(x))_{i} = i^{th} output$$

$$J(\Theta)=-\frac{1}{m} \left [\sum_{i=1}^{m}\sum_{k=1}^{K}y_{k}^{i}log((h_{\Theta}(x^{i}))_{k}) + (1-y_{k}^{i})log(1-(h_{\Theta}(x^{i}))_{k}) \right] + \frac{\lambda}{2m} \sum_{l=1}^{L-1} \sum_{i=1}^{s_{l}} \sum_{j=1}^{s_{l+1}} (\Theta_{ji}^{l})^{2}$$

与逻辑回归不同的是这里的输出 $h_{\Theta}(x)$ 是一个 $K$ 维向量，其中 $K$ 代表类别的数量。并且由于这个式子引进了更多的求和与下标所以看着好像很复杂，但事实上这只是对逻辑回归损失函数的简单扩展。

可以将 $J(\Theta)$ 分为两大部分来看，第一大部分为：

$$-\frac{1}{m} \left [\sum_{i=1}^{m}\sum_{k=1}^{K}y_{k}^{i}log((h_{\Theta}(x^{i}))_{k}) + (1-y_{k}^{i})log(1-(h_{\Theta}(x^{i}))_{k}) \right]$$

这一项是损失函数的主体，这里事实上就是对每个样例的每个输出计算损失并进行统一求和，并且由于每个输出都是一个 $K$ 维向量，因此还需要对向量中的每个元素再进行损失求和。

Python3 配置 OpenCV3

发表于 2017-01-02   |   分类于 OpenCV 学习记录   |     |   阅读次数

说明：更新于 2017/01/02 ，添加了 Linux 中编译安装 OpenCV 的内容。

windows 下安装 opencv

从 opencv 3.x 开始，opencv 其实就已经可以支持 python 3 了，但是官方给出的编译版本还只支持 python 2.7 ，所以如果想要给 python 3 安装 opencv 模块，我们就得自己编译源码，但是这个过程相对繁琐，还容易出错。

这里提供一个非官方的下载地址：

http://www.lfd.uci.edu/~gohlke/pythonlibs/#opencv

该网站提供了众多在 windows 平台下的第三方已编译 python 包扩展下载。

进入网页，选择一个合适自己的版本 opencv_python-xxx.whl 下载。

比如，我使用 python 3.5 ，所以选择下载 ：

opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win_amd64.whl

amd64 适用所有 64-bit 的操作系统。

下载好之后，使用 CMD 进入 .whl 文件所在目录，执行（以 opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win_amd64.whl 为例）：

pip install opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win_amd64.whl

进行安装。

Cmder 中 ls 命令中文乱码问题解决

发表于 2016-12-26   |   分类于 技术汇   |     |   阅读次数

Cmder 是 Windows 平台中一个非常好用的第三方 cmd 。不仅颜值非常高还集成了 git ，并且支持部分 Linux 指令，比如 ls、 cp 等。

它的安装也非常简单，直接从官网下载所需要的压缩包，解压之后配置路径名到环境变量便可使用。

但是目前最新版的（v 1.3.2）Cmder 依然存在适用 ls 或者 cat 指令中文显示乱码的问题。

上网搜了一下给出的方案都是修改 Cmder 安装路径下的 ./config/aliases 文件，添加 ls=ls --show-control-chars -F --color $*。

这种方法在 v 1.3.0 之前的还可用，但是在 v 1.3.0 之后的 aliases 文件已经被 user-aliases.cmd 取代了，并且默认添加了 ls=ls --show-control-chars -F --color $* ，所以这个方法对 v 1.3.0 只有的版本不再适用。
