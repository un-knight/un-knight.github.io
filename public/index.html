发表于 2016-11-06   |   分类于 python   |  

由于国内的网络环境问题，导致我们在使用 pip 安装一些模块软件的时候可能出现 time out 错误。遇到这种情况就让人相当恼火了，不过我们可以采取更改下载源的方式，使 pip 从国内的源下载来解决。

常见的国内镜像源有：

  • 豆瓣：http://pypi.douban.com/
  • V2EX：http://pypi.v2ex.com/
  • 华中理工：http://pypi.hustunique.com/
  • 山东理工：http://pypi.sdutlinux.org/
  • 中科大：http://pypi.mirrors.ustc.edu.cn/
