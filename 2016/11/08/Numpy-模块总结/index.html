hstack(list)

用于将多个矩阵进行水平方向上的拼接。传入的参数为 list 列表，即将需要拼接的矩阵组装成一个列表，如 [array1, array2, array3]

linspace(start, end, num=50, endpoint=True)

用于创建一维的等差数列， start 为起点，end 是终点， num 是点的个数， endpoint 指定是否包含终点。

注意 endpoint 值将影响最终产生数组的步长
