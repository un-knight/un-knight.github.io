第一章：简介

发表于 2016-11-13   |   分类于 Machine Learning   |     |   阅读次数

说明：这是我在学习机器学习过程中所做的总结与思考，毕竟才刚开始学习 ML 而且能力也有限，所以笔记中难免有错，欢迎交流指正。

首先奉上两位 ML 的大牛对于 _Machine Learning_ 的定义来镇楼。

  • “Field of study that gives computers the ability to learn without being explicitly programmed” – Arthur Samuel
  • “A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.” – Tom Michel

机器学习主要分为三大类算法：

  • 监督学习
    • 使用带标签数据进行训练，解决回归与分类问题。
  • 无监督学习
    • 通过不带标签的数据训练模型，让算法自动从数据中发现某种特定模式。
  • 增强学习

    一、监督学习

    监督学习通常用于解决回归（regression）和分类（classification）问题。

1.1 回归

那么什么是回归问题呢？这个很简单。考虑一种最简单的情况，在直角坐标系中给出一系列散点，通过将这些散点以平缓的线条连接起来我们就可以得出一条曲线的一部分。那么回归问题就是要利用目前已经获得的局部散点去计算出曲线方程来尽可能地全局或者局部拟合出目标曲线。最终来实现预测的目的（比如给出一个原先数据集中不存在的 x 值，要预测出相应的 y 值）。

回归模型在生活中有着很广泛的应用，比如预测房价、股市走向、天气和成绩等，只不过在现实生活中一件事情的确定往往是在很复杂的因素共同作用下形成的，这就需要我们不仅是通过数学分析方法还需要结合经验来确定最有价值的干扰因素，从而在保证准确率的前提下，减少参与统计分析的因子，以降低分析的复杂度。

1.2 分类

分类问题顾名思义就是实现分类。这与我们在生活中挑选水果的时候或者在淘宝店上买东西时候的过程很相似。比如，我们在淘宝上决定是否购买某一商品时通常需要综合考虑以下因素：

  • 价格
  • 商家评级
  • 好评率
  • 差评率
  • …

在每个人心中这些因素都有各自的权重，当我们做了一个综合考量之后我们会在心中给这家淘宝店的商品打个分，当这个分数高于心中的及格线之后也许你就会开始有剁手购买的欲望，反之你可能就对此感到失望而放弃购买。

而算法在实现分类问题上所做的事与人的决策过程十分相似。再举个更加简单的例子，比如你很喜欢吃苹果但是你并不知道哪种苹果好吃，于是你挑选了颜色作为判断的依据。思维严谨的你到超市买了一些苹果，它们的颜色依次从青色逐渐变化到红色，然后你开始依次品尝并做了相应的记录。最后你发现被你标记了好吃的苹果颜色都是偏向红色这一边，而不好吃的都是偏向绿色那一边，于是你得出了一个结论，颜色比较红的苹果比较好吃。但是你生性比较懒，懒得自己去判断超市里边那些苹果是好吃的，于是你将之前做实验所得到的数据交给机器来学习，从而让机器来帮助你挑选出你喜欢吃的苹果。

1.3 回归与分类的区别

回归与分类都是监督学习，但是回归解决的连续问题而分类解决的是离散问题。但是这个并不是绝对的，在某些情况下回归也能用来解决分类问题。

1.4 总结

  1. 能得出数学意义上“正确”的结果
  2. 回归问题（regression）
  3. 分类问题（classification）

二、无监督学习

在无监督学习中我们使用无标签的数据进行训练，让算法自动从数据中发现某种特定的结构模式。其中一种方法就是聚类算法。

  • Google news
  • 社交网络分析
  • 天文数据分析
