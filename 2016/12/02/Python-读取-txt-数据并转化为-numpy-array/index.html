Python 读取 txt 数据并转化为 numpy.array

发表于 2016-12-02   |   分类于 python   |     |   阅读次数

在机器学习或者数据挖掘的学习过程中会发现很多数据都会存储在 CVS 文件中，但是这些数据也可以存储在 txt 文件中。

想要使用 python 对这些数据进行处理的时候却不知道如何快捷地从 txt 文件中读取数据怎么行，这里我将分享我的处理方法。

到了后来才发现 numpy 也提供了一个强大的格式化读取 txt 文件的接口 numpy.loadtxt 。

默认情况下是以空格为分隔符，读取为浮点型数据。

当然也可以通过参数 delimiter 自定义分隔符，参数 usecols 指定读取的列（索引从 0 开始）以及执行 unpack 操作。
